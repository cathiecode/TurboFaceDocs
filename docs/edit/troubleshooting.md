# TurboFace Editで困ったときは
まずはこれを試してみてください。

## なんか全然動かない
- 「VRChat SDK」→「Reload SDK」をしてからもう一度試してみてください。
- Unityを閉じて、VCCから削除して、インストールしなおしてみてください。

## スライダーを変更してもモデルが動かない
- プレビューを有効化し忘れていませんか？TurboFace Editウィンドウの一番上にあるボタンで有効化することができます。

## フォルダ表示が壊れている
フォルダの区切り文字の自動検出に失敗したのかもしれません。「ブレンドシェイプエディタの設定」→「フォルダの区切り文字」から設定を調整してみてください。

## ブレンドシェイプピッカーが上手く動かない
モデルによっては、ブレンドシェイプピッカーがうまく機能しない場合があります。
「ブレンドシェイプエディタの設定」→「動いたと判断する距離」を調整してうまくいく値があるか試してみてください。
