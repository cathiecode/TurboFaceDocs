# TurboFaceへようこそ！

TurboFaceは、VRChat向けの表情アニメーションを総合的に扱うことができるエディタ拡張です。表情アニメーションを作成・編集・管理・設定することができます。
以下の二つの機能は、同時に使うことも、片方だけ使うこともできます。**もし片方が気に入らなければ、もう片方だけを使う**ことも可能です。
- TurboFace Edit: 爆速で表情アニメーションを作成
- TurboFace Setup: 表情アニメーションの設定をアバターに非破壊で適用。

## TurboFace Edit
TurboFace Editは、何百もの表情用ブレンドシェイプがあるアバターで高速に表情アニメーションを作成できるように設計された表情エディタです。

### 主な機能
- 表情アニメーションの作成・編集・保存
- 表情アニメーションをプレビューしながら編集
- マウスホバーでブレンドシェイプの挙動を確認
- 名前からブレンドシェイプを探す
- 似た名前からブレンドシェイプを探す
- 動かしたいメッシュをクリックしてブレンドシェイプを探す
- ブレンドシェイプをフォルダに分けて表示する

## TurboFace Setup(開発中機能)
TurboFace Setupは、表情用アニメーションを非破壊でアバターに適用するためのツールです。

### 主な機能
- アバターにすでについている表情をインポートする
- ジェスチャに対応して表情を変更する
- メニューから表情を変更する
- まばたきアニメーションを設定する

まだドキュメンテーションはありません。興味があれば、Edit→Project Settings→Project→TurboFace→Enable experimental features から有効化できます。
